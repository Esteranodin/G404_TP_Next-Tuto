# Tuto Next.js

## START

* Install the project's packages : 

    ```bash
    pnpm i
    ```

* Starts the development server : 

    ```bash
    pnpm run dev
    ```

* Builds the app for production : 

    ```bash
     pnpm run build
    ```

* Runs the built app in production mode : 

    ```bash
     pnpm start
    ```

    
*  Don't forget .env with database connexion
